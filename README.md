# minishell
![demo](demo.png)
## Description
simple shell like bash

## Requirement
* GCC
* GNU Make

## Installation
```shell
git clone --recursive git@github.com:tkomatsu/minishell.git
cd minishell
make
```

## Usage
```shell
./minishell
```
